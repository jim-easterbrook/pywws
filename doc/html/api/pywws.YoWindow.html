pywws.YoWindow¶

Generate YoWindow XML file.

usage: python pywws/YoWindow.py [options] data_dir output_file options are:

-h or –help display this help -v or –verbose increase amount of reassuring messages

data_dir is the root directory of the weather data output_file is the YoWindow XML file to be written
